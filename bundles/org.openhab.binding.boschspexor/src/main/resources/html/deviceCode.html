Authorization of openHAB to for binding Bosch spexor

To use the openHAB with the Bosch spexor please authorize your openHAB with spexor.

Please register this service with the Bosch spexor app. Therefore please proceed with following steps:

  • Open the Bosch spexor app.
  • Go to: Account
  • click on "spexor API"
  • Register new service
  • option 1) scan the current QRCode
  • option 2) enter the device code into the app
  • proceed with the app

This openHAB service is currently UNAUTHORIZED to communicate with the spexor API
