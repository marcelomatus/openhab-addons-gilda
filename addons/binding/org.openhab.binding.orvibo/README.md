# Orvibo Binding

This binding integrates Orvibo products that communicate using UDP. Primarily this was designed for S20 Wifi Sockets but other products using the same protocol may be implemented in future.

## Supported Things

* S20 Wifi Sockets

## Discovery

This binding can automatically discover devices that have already been added to the Wifi network.  Please see the check your instruction manual or the help guide in the app for instructions on how to add your device to your Wifi network.

## Binding Configuration

This binding does not require any special configuration.
